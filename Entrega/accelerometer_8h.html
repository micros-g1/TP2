Detailed Description

Accelerometer interface.

Author
Grupo 1 Labo de Micros
Date
1 Octubre 2019

This interface is prepared to handle the communication with the magnetometer and accelerometer of the freedom board. The interface will recollect new data from the sensors in a periodic fashion, so that if the user asks for data faster than the refresh rate of the sensors, the data he/she/Apache Helicopter will be the same in at least two consecutive calls
