Detailed Description

ID+DLC+Data: SIDH/SIDL/EID8/EID0/DLC and Buffer.

Represents the whole buffer (without configuration register)
Buffer Id + DLC + Buffer data.
It is important to note that Id+DLC info format is slightly different for Transfer buffer and receive buffer:
Transfer: "Remote Transfer Request" is given by .dlc.rtr Receive: "Remote Transfer Request" is given by .id.ssr if .id.eid == true, else it is given by .dlc.rtr
